# NfeUi 牛卡福PC端UI组件库。

在线预览地址：https://nfeui.nucarf.cn  

Vue3版本 
## 特性

* 使用VUE 3语法
* 支持按需引用
* 详尽的文档和示例
* 支持 TypeScript

## 组件

* 列表搜索组件（待开始）
* table组件（待开始）
* 。。。。

* chrome的支持 查看->[caniuse](https://caniuse.com/es6-module-dynamic-import)
